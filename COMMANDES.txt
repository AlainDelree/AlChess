COMMANDES NICLINK
=================
  Pour jouer :
  bash ~/AlChessPlay/jouer.sh


LANCER LE JEU EN TRAVAIL
-------------
cd ~/NicLink
source venv/bin/activate
cd ~/NicLink && python -m nicsoft.web


LANCER LE JEU TIME DEBUG
-------------------------
cd ~/NicLink
source venv/bin/activate
NICLINK_LOG=DEBUG python -m nicsoft.web

MODE CONFIG ALEATOIRE---------------------------------------
---------------------
cd ~/NicLink
source venv/bin/activate
NICLINK_TEST=random python -m nicsoft.web


TIME LOG FILTRE
---------------
cd ~/NicLink
source venv/bin/activate
python -m nicsoft.web 2>&1 | grep -E "TIMING|WAIT_FISH"

DISTRIBUTION NOUVEAU PC
----------------------
pip install git+https://github.com/AlainDelree/AlChess.git

SAUVEGARDES
-----------
Backup normal :
python -m nicsoft.utils.backup_manager

Backup épinglé (jamais supprimé automatiquement) :
cd ~/NicLink
source venv/bin/activate
python -m nicsoft.utils.backup_manager --pin --label "Avant-ReArchitecture"

Lister les backups :
python -m nicsoft.utils.backup_manager --list

Simuler sans rien créer :
python -m nicsoft.utils.backup_manager --dry-run


Les backups sont sauvegardés dans : ~/NicLink_backups/
  - normal/   : rotation automatique (max 5)
  - pinned/   : jamais supprimés automatiquement
